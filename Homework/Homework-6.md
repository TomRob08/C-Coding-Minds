# Homework 6

Create a program that uses two user generated numbers and prints the power of the numbers. The first number is the base and the second number is the exponent. Solve this problem using a loop.

### Input:
```
2 2
```

### Output:
```
4
```

### Input:
```
5 3
```

### Output:
```
125
```

### Input:
```
3 4
```

### Output:
```
81
```
